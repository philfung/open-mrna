# Interactive Workflow Explorer

This is a React-based interactive diagram built with [React Flow](https://reactflow.dev/) to visualize the mRNA cancer vaccine pipeline.

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Run in development mode:**
    ```bash
    npm run dev
    ```
    This will start the Vite server. Visit the URL provided in the terminal (usually `http://localhost:5173`).

## Building for Production

To build the static version of the app:
```bash
npm run build
```
The output will be generated in the `../docs` directory.
